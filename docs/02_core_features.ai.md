# Core Features
## Key Advantages
### 1. 亿级3DGS点云原生支持
- 内置自适应 LOD，支持亿级以上大场景流式加载

### 2. 自研 SPX 高压缩格式
- 相比 PLY，文件体积减少 **95%+**，加载速度显著提升
- 原生支持流式渲染与 LOD 分块瓦片
- 配套跨平台转换工具：**gsbox**

### 3. 跨平台深度优化（1.5 亿点场景实测）
| 设备类型 | 渲染性能 |
|---|---|
| 桌面端（独显） | 稳定 **60 FPS** |
| 轻薄本（集显） | 稳定 **30 FPS** |
| 主流手机 | **25–40 FPS**，温控优秀 |
| 老旧手机 | **7–15 FPS+**，稳定无崩溃 |

缓存可配，内存占用稳定无泄漏

### 4. 全格式兼容
- 原生支持：`.ply`、`.splat`、`.spx`、`.spz`、`.sog`
- 扩展支持：带 `KHR_gaussian_splatting` 扩展的 `.glb` 文件

### 5. 开箱即用商用工具
- 标注、距离/角度测量
- 自定义水印
- GIS 底图集成
- 单模型独立配置（`.meta.json`）
